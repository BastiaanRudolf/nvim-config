## Setup

This repo assumes you're running MacOS.

### Prerequisites

1. Neovim
```zsh
brew install neovim
```
2. ripgrep
```zsh
brew install ripgrep
```
3. (optional) A nerd font
```zsh
brew install font-fira-code
```
4. (optional) tmux
```zsh
brew install tmux
```

### Installation

1. Clone this repo to your `~/.config/nvim/` folder.
